**파일:** `AmadooWidget/TimetableWidget.swift` (신규)

**기능:**
- **크기:** systemLarge
- **레이아웃:**

#### 3.1 AppDelegate - 카테고리 정리 로직 개선

**변경 내용:**
```swift
func cleanupInvalidCategories() {
    let context = persistentContainer.viewContext
    let fetchRequest = NSFetchRequest<NSManagedObject>(entityName: "Category")

    context.perform {
        for category in categories {
            var shouldDelete = false

            if let name = category.value(forKey: "name") as? String {
                // "Unknown", "마이그레이션 카테고리*" 삭제
                if name.hasPrefix("마이그레이션 카테고리") || name == "Unknown" {
                    shouldDelete = true
                }
            } else {
                // name이 nil인 카테고리도 삭제
                shouldDelete = true
            }

            if shouldDelete {
                context.delete(category)
                print("🗑️ 잘못된 카테고리 삭제: \(category)")
            }
        }

        // 저장 및 위젯 업데이트
        try? context.save()
        WidgetCenter.shared.reloadAllTimelines()
    }
}
```

        let storeURL = appGroupURL.appendingPathComponent("NewCalendar.sqlite")

## 💡 개발 노트 & 베스트 프랙티스

### iOS 위젯 개발
- iOS 17+에서는 `.containerBackground(for: .widget)` API가 **필수**입니다.
- 위젯은 최대 15분마다 자동 업데이트됩니다 (시스템 제한).
- App Group을 통한 데이터 공유는 실시간 동기화가 아니므로, 위젯 타임라인 정책을 적절히 설정해야 합니다.
- 위젯에서 복잡한 연산은 피하고, 미리 계산된 데이터를 제공하는 것이 좋습니다.

### 햅틱 피드백
- 햅틱 피드백은 시뮬레이터에서 동작하지 않으며, **실제 기기에서만** 확인 가능합니다.
- `UIImpactFeedbackGenerator` 사용 시 `prepare()` 메서드를 미리 호출하면 지연 시간을 줄일 수 있습니다.

### iTunes Lookup API
- Bundle ID가 정확해야 합니다.
- 앱스토어에 출시된 앱만 조회 가능합니다.
- 심사 중이거나 출시 전 앱은 `resultCount: 0`을 반환합니다.
- API 응답 시간은 보통 1-3초 정도 소요됩니다.

### CloudKit 동기화
- NSPersistentCloudKitContainer 사용 시 자동으로 동기화됩니다.
- 초기 동기화 시 시간이 걸릴 수 있습니다 (1-2분).
- 네트워크 상태에 따라 동기화 시점이 달라질 수 있습니다.
- 동기화 충돌 시 "최신 변경 우선" 정책 적용됩니다.

### 성능 최적화
- 위젯은 메모리 제한이 있으므로 (30MB), 대량의 데이터를 로드하지 않도록 주의합니다.
- CoreData fetch는 필요한 데이터만 가져오도록 predicate를 사용합니다.
- 이미지는 가능한 한 작게 유지합니다 (SF Symbols 권장).

## 🔐 보안 및 권한

### 추가된 권한
1. **App Groups** (`group.Simoni.Amadoo`)
   - 용도: 앱-위젯 간 데이터 공유
   - 데이터: CoreData, UserDefaults

2. **iCloud** (`iCloud.Simoni.Amadoo`)
   - 용도: CloudKit 동기화
   - 데이터: 일정, 카테고리, 시간표

3. **Network** (iTunes Lookup API)
   - 용도: 앱 버전 체크
   - 데이터: Bundle ID, 버전 정보

### 개인정보 처리
- iTunes Lookup API는 사용자 개인정보를 전송하지 않습니다.
- CloudKit은 사용자의 iCloud 계정으로 암호화되어 저장됩니다.
- 위젯은 로컬 데이터만 사용하며, 외부로 전송하지 않습니다.
